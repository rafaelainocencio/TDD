# Testes Unitários
## Projeto de Estudos sobre Testes Unitários, utilizando XUnit.
## Temas Abordados:
* Estrutura de teste AAA (Arrange - Act - Assert)
* TDD (Test-Driven Development)
* TDD: Red - Green - Refactor | Falhar - Corrigir - Melhorar
* Teste de 4 passos SEVT (Setup - Exercise - Verify - Teardown)
* Teste de 5 passos SAAAT (Setup - Arrange - Act - Assert - Teardown)
* xUnit
* xUnit - Theory: InlineData, MemberData, ClassData
* xUnit - Collection (Paralelism
